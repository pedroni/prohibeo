Prohibeo is a Chrome extension for blocking distracting websites and hiding the parts
of a page that make it harder to stay focused.

This starter post exists for two reasons:

1. It gives the landing page a working MDX-powered blog route.
2. It shows the kind of lightweight content workflow Astro is good at.

## What Prohibeo does today

- Blocks entire sites
- Supports always-on and scheduled blocking
- Includes built-in YouTube cleanup controls
- Lets you add custom CSS selectors for your own distraction filters

## Why Astro for the site

Astro is a good fit here because the site is mostly static content, and the project does
not need the extra framework surface area of a heavier application setup.

That means you get:

- fast static pages
- strong SEO fundamentals
- clean MDX content authoring
- very little runtime JavaScript
